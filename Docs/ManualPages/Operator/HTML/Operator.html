On the Subject of Operators

You're pretty sure these rotary dials haven't been used since the 70's. You get the sense this might actually still be connect to a telephone.

  • The LED sequence defines the start point in the dialing table.
  • If the bomb had a 2FA code, then read left to right (→), otherwise top to bottom (↓).
  • If the bomb serial number does not have a vowel, then reverse the code.
  • Dial the five digit number.
