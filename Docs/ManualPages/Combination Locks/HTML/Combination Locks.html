On the Subject of Combination Locks

This looks like a combination lock. I thought I was disarming this bomb, not unlocking it.

Like a typical combination lock, this requires 3 numbers to unlock.

Turn the dial to the first number, push on the dial, turn to the second number and push the dial. Finally, turn it to the last number and push the dial. That will unlock it! If sequential numbers in the code are the same, perform a full revolution back to the same number.

Which direction to turn:
  • Before turning the dial, each digit of the initial number displayed must be summed, if it can be evenly divided by three then the initial direction should be clockwise, otherwise anti-clockwise.
  • Each subsequent direction should alternate.
  • The dial must not be turned both directions, if a number is missed perform another revolution in the same direction.
How to decode the combination:

Each number ranges from 0-19. Refer to the table to determine each number.

First number:
  1. Add the first two digits of the two factor code together. If there is no two factor code, use the last digit of the serial number plus the number of solved modules.
  2. If the result is over 19, keep subtracting 20 until the result is between 0-19.
Second number:
  1. Add the last two digits of the two factor code together. If there is no two factor code, use the number of modules on the bomb (including needy modules and timer).
  2. If the result is over 19, keep subtracting 20 until the result is between 0-19.
Third number:
  1. Add the first two numbers together.
  2. If the result is over 19, keep subtracting 20 until the result is between 0-19.

On incorrect entry the lock will reset and the whole process must begin again.
